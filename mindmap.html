3 — Special Rules & Edge Cases
🔒
CSEC 181 → CSEC 281 Prerequisite (Net-Sec, Offensive Security track)
CSEC 281 cannot be enrolled until CSEC 181 is completed with a grade of C or higher. Cannot be taken before, during, or at the same time as CSEC 181.

Note: CSEC 281B = CIT 274 (same course, different code) — this is not the same as CSEC 281.
🔄
CHFI Substitution Path (DF AAS & CA old-map students)
If CSEC 112B / 116B / 212B are still needed (not yet completed), the recommended substitute path is the full CHFI sequence — taken in strict sequential order:
1CSEC 120 → 2CSEC 121
↓ complete both 100-level courses before enrolling in 200-level ↓
3CSEC 220 → 4CSEC 221
These are also required core in the 2026–27 map — a strong reason to switch maps.
⚠️
3-Scenario Deactivated Course Logic (DF Programs)
Applied to: CSEC 112B, CSEC 116B, CSEC 212B

Scenario A — None completed yet → Recommend CHFI sequence as substitute.
Scenario B — Some completed, some still needed → Two options: CHFI sequence (full re-do) OR individual substitution with Prof. Morningstar.
Scenario C — All completed → All count as elective credits toward degree. Optional CHFI for certification.
⏰
CSEC 290B — Last Offered Fall 2026 (AAS only)
AAS students only. CA students must NOT enroll.

If the student has not yet completed CSEC 290B, they must register for Fall 2026 immediately — this is the final offering. After Fall 2026 this course will no longer be available under any circumstances.

Handled separately from the deactivated courses (112B/116B/212B) — different urgency, different message.
↔️
Course Code Equivalencies
Old code → New code (same course):
CSEC 110B = CSEC 110
CSEC 281B = CIT 274
CSCO 230B = CSCO 230
CSEC 104B = CSEC 104 (Net-Sec 27–28)
CSEC 112 = CIT 112 (crosslisted)

⛔ Common confusion: CSEC 281B ≠ CSEC 281. They are different courses.
💳
4-Credit Course Exceptions
Most courses are 3 credits. The following are 4 credits:

CIT 114 / CSEC 114, CSCO 120, CSCO 121, CSCO 130B, CSCO 205B, CSCO 220, CSCO 230, CSCO 230B, CSCO 480, CSCO 482, CSCO 484, CSCO 105B.

Tuition is charged per credit — students should be aware of the higher cost.
👤
Professor Morningstar — Individual Substitution
Available when a student has partially completed discontinued DF courses (Scenario B). The student meets individually with Prof. Morningstar to identify suitable CIT, CSEC, or CSCO substitute courses.

Student must bring transcript and current course plan. Approved substitutes must be CIT, CSEC, or CSCO courses.
📊
Map Comparison Feature
Old-map students can run their same completed courses against the 2026–27 map to see how many credits would carry over without re-taking anything.

The comparison is read-only — switching maps officially requires an advisor appointment. The tool displays which courses carry ✅ and which don't ❌.
